# 哈哈镜相机小程序

一个有趣的微信小程序，可以拍照并应用各种哈哈镜效果。

## 功能特点

- 前后摄像头切换
- 多种哈哈镜效果
  - 正常模式
  - 胖胖效果
  - 瘦瘦效果
  - 高高效果
  - 矮矮效果
- 照片保存功能
- 用户授权登录

## 技术实现

- 使用微信小程序原生开发
- 使用 camera 组件实现拍照功能
- 使用 canvas 实现图片变形效果
- 针对性能优化：
  - 图片压缩处理
  - 限制图片尺寸
  - 优化渲染性能

## 使用说明

1. 首次使用需要授权相机权限
2. 点击"开始拍照"进入相机界面
3. 可以通过右侧按钮切换前后摄像头
4. 左侧按钮可以切换不同的哈哈镜效果
5. 中间大按钮用于拍照
6. 拍照后可以：
   - 保存照片到相册
   - 重新拍摄
   - 切换效果重新处理

## 安装和运行

1. 克隆项目到本地
2. 使用微信开发者工具打开项目
3. 在项目设置中配置自己的 AppID
4. 编译运行即可

## 注意事项

- 需要授权相机权限才能使用
- 建议在光线充足的环境下使用
- 图片大小限制在2MB以内
- 某些机型可能会有兼容性问题
